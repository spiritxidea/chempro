# Knowledge Center CSV Upload

Replace `knowledge.csv` with updated content, then build and deploy the site.

Required columns:

```csv
header,subject,body,image
```

Image can be:

- A full URL, such as `https://example.com/image.jpg`
- A public path, such as `/assets/knowledge-ro.png`
- A file name placed in this folder, such as `article-01.png`
